# AI Smart Surveillance System

🔗 Live Demo: https://ai-smart-surveillance-system-6ixt.vercel.app/

AI Smart Surveillance System is an intelligent web-based platform that uses computer vision and machine learning to transform traditional CCTV monitoring into proactive, real-time security.

## 🚀 Features

- Real-time object detection (people, vehicles, anomalies)
- Motion tracking and incident extraction
- Facial recognition support
- Live monitoring dashboard with alerts
- False positive reporting system
- Privacy-focused local inference

## 🛠 Tech Stack

**Frontend:** React (Vite), HTML, CSS, JavaScript  
**Backend:** Node.js / Flask  
**AI/ML:** OpenCV, Computer Vision models  
**Deployment:** Vercel / Render
